# 📝 Todo API – NestJS

API backend de gestion de todos (tâches), développée avec **NestJS**.  
Projet pédagogique visant à comprendre la structure d’un backend moderne, et la mise en place d’un CRUD propre.

---

## 🚀 Stack technique

- **Node.js** ≥ 20
- **NestJS** v11
- **TypeScript**
- **dotenv**

---

## 📁 Structure du projet

src/
├── app.module.ts
├── app.controller.ts
├── main.ts
└── todo/
├── todo.module.ts
├── todo.controller.ts
├── todo.service.ts
└── todo.entity.ts
