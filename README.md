# 📝 Todo API – NestJS

API backend de gestion de todos (tâches), développée avec **NestJS**.  
Projet pédagogique pour comprendre la structure d’un backend moderne et mettre en place un CRUD simple **sans base de données**.

---

## 🚀 Stack technique

- **Node.js**
- **NestJS**
- **TypeScript**

---

## 📁 Structure du projet

src/
├── app.module.ts
├── app.controller.ts
├── main.ts
└── todo/
├── todo.module.ts
├── todo.controller.ts
└── todo.service.ts
